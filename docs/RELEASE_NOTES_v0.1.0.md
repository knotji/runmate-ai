# RunMate AI v0.1.0 — Beta

ขอบคุณที่ร่วมทดสอบ RunMate AI ในช่วง Beta 🙏
แอปนี้ยังอยู่ระหว่างพัฒนา ความคิดเห็นของคุณช่วยให้เราปรับได้ตรงจุดที่สุด

---

## สิ่งที่ทำได้ใน v0.1.0

**โค้ชรู้จักสภาพร่างกายคุณจริง ๆ**
วันไหนนอนน้อยหรือฟื้นตัวยังไม่ดี โค้ชจะพูดอย่างนุ่มนวล ไม่กดดัน
และแนะนำให้ลดโหลดหรือพักก่อน ไม่ใช่ push ให้ซ้อมหนัก

**Today, Coach และ Race ใช้ข้อมูลชุดเดียวกัน**
ไม่ว่าจะเปิดหน้าไหน คำแนะนำจะสอดคล้องกัน — ตามสถานะ recovery จริงของวันนั้น

**Report มี Weekly Insight**
ทุกสัปดาห์จะมีสรุปการนอน การวิ่ง และการฟื้นตัว เพื่อให้เห็นแนวโน้มได้ชัดขึ้น

**Auto Sync โปรไฟล์ — เคารพข้อมูลที่แก้เอง**
ระบบวิเคราะห์ประวัติการวิ่งและการนอนเพื่อเติมโปรไฟล์ให้อัตโนมัติ
แต่ถ้าคุณแก้ค่าไหนเองไว้ ระบบจะไม่แตะ

**หน้า Upload ชัดขึ้น**
แต่ละประเภทมีคำอธิบายว่าข้อมูลนี้ใช้ทำอะไร บันทึกอะไร และเชื่อมกับโค้ชอย่างไร

**พร้อมใช้งานแบบ PWA**
ติดตั้งไว้บนหน้าจอมือถือได้เลย ใช้ได้เหมือนแอปปกติ
มีแบนเนอร์แจ้งเตือนเมื่อออฟไลน์ และหน้าว่างรองรับผู้ใช้ใหม่ที่ยังไม่มีข้อมูล

---

## สิ่งที่ยังไม่มีใน Beta นี้

- ยังไม่รองรับการ import ข้อมูลจากนาฬิกาโดยตรง (ใช้ภาพถ่ายหรือ screenshot แทน)
- ยังไม่มีการแจ้งเตือน (push notification)
- Race Plan ยังเป็นแผนรายสัปดาห์ ยังไม่ใช่รายวันแบบละเอียด

---

## ฝากบอกด้วยถ้า…

- คำแนะนำโค้ชดูไม่ตรงกับวันนั้น
- ข้อความภาษาไทยอ่านแล้วงง หรืออ่านแล้วดูแปลก
- หน้าไหนโหลดช้า หรือปุ่มไม่ตอบสนอง
- มีข้อมูลที่อยากบันทึกแต่ยังไม่มีหน้า

ส่งผ่าน Line ได้เลย หรือจะ screenshot แล้วส่งมาก็ได้ครับ 🙌
